## ⚙️ Установка

1. Закройте игру, если она запущена;

2. Скачайте архив (zip/rar) со [страницы последнего релиза](https://github.com/sharewyd/the_convenience_store_ru_localization/releases/latest);

3. В целях предосторожности (на случай нестабильной работы игры) скопируйте следующие файлы в отдельную папку на рабочем столе. Из папки "The Convenience Store Yakin Jiken_Data" (обычно это: `C:\Program Files (x86)\Steam\steamapps\common\The Convenience Store Yakin Jiken\The Convenience Store Yakin Jiken_Data\`):
    ```cmd
    level0
    level1
    resources.assets
    sharedassets0.assets
    ```
    
4. Распакуйте содержимое архива в папку с установленной игрой, подтверждая замену файлов (обычно это: `C:\Program Files (x86)\Steam\steamapps\common\The Convenience Store Yakin Jiken\The Convenience Store Yakin Jiken_Data\`);

5. Запустите игру.

## ℹ️ Описание модификации

Данная модификация полностью заменяет английский текст на русский.
